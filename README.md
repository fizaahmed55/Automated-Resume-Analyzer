# Automated Resume Analyzer

Automated Resume Analyzer is a Java-based application developed to analyze resumes and match them with suitable job requirements. The project helps in identifying relevant skills, experience, and keywords from a resume and comparing them with job-related criteria.

## Features

- Analyze resume content
- Extract important skills and keywords
- Match resume data with job requirements
- Provide job-matching assistance
- Java-based desktop/application project

## Technologies Used

- Java
- Eclipse IDE
- Maven
- Git
- GitHub
